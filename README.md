# 盗賊ゲーム 運営補助ツール

ブラウザで動作する、ゲームマスター専用の「盗賊ゲーム」運営補助Webアプリです。

プレイヤー全員が操作するゲーム本体ではなく、ゲームマスターがPC上で進行管理・行動入力・ポイント計算・結果表示を行うための補助ツールです。

## 公開ページ

GitHub Pagesで公開して使用します。

## 対応環境

* PCブラウザ向け
* GitHub Pages対応
* HTML / CSS / JavaScriptのみで動作
* サーバー不要
* インストール不要

## 推奨人数

* 推奨人数：5〜7名
* プレイ可能人数：4〜8名

## 主な機能

* プレイヤー人数設定
* プレイヤー名登録
* 初期ポイント設定
* ラウンド数設定
* ラウンド進行管理
* 各プレイヤーの行動入力

  * 奪う
  * 守る
  * 貯める
* 「奪う」選択時のみ対象プレイヤーを指定
* 入力確認画面
* 確定後のポイント自動計算
* 喧伝システム
* 現在ランキング表示
* 最終順位表示
* ラウンド履歴表示
* オートセーブ
* 保存データからの再開
* Discord投稿用結果テキスト生成
* Discord用テキストのコピー

## 基本的な使い方

1. 設定画面でプレイヤー人数、初期ポイント、ラウンド数を設定します。
2. プレイヤー名を入力します。
3. 「新しくゲーム開始」を押します。
4. ラウンド画面で各プレイヤーの行動を入力します。
5. 「入力内容を確認」を押します。
6. 確認画面で内容を確認します。
7. 問題なければ「この内容で確定」を押します。
8. 結果画面で処理結果とランキングを確認します。
9. 必要に応じて、喧伝するプレイヤーを選択します。
10. 喧伝を行う場合は「喧伝を確定」を押します。
11. 次のラウンドへ進みます。
12. 最終ラウンド終了後、最終順位とDiscord投稿用テキストを確認します。

## 行動一覧

### 奪う

指定したプレイヤーからポイントを奪います。

対象プレイヤーの行動によって結果が変わります。

* 対象が「貯める」だった場合：奪った側 +10P
* 対象が「守る」だった場合：奪った側 -5P
* 対象も「奪う」だった場合：変動なし

### 守る

自分の財産を守ります。

* 1人以上から「奪う」の対象になった場合：+5P
* 誰にも狙われなかった場合：-1P

複数人から狙われても、守る成功による +5P は1回のみ適用されます。

### 貯める

安全にポイントを蓄えます。

* 基本：+5P
* 1人以上から「奪う」の対象になった場合：追加で -10P

つまり、貯めたプレイヤーが誰かに狙われた場合は、
+5P と -10P が両方適用され、合計で -5P になります。

複数人から狙われても、貯める失敗による -10P は1回のみ適用されます。

## ポイント処理について

各ラウンドにおいて、ポイントの増減は以下の方針で処理されます。

* 「奪う」側の成功・失敗は、奪う行動ごとに処理されます。
* 「守る」側の成功は、複数人から狙われても1回のみ処理されます。
* 「貯める」側の失敗は、複数人から狙われても1回のみ処理されます。
* 「守る」を選び、誰にも狙われなかった場合は -1P されます。
* 「貯める」を選び、誰にも狙われなかった場合は +5P されます。

## 処理例

### 例1

```text
A：貯める
B：奪う → A
C：守る
D：奪う → C
```

結果：

```text
A：+5P -10P = -5P
B：+10P
C：+5P
D：-5P
```

### 例2：複数人が同じ相手を奪う場合

```text
A：貯める
B：奪う → A
C：奪う → A
```

結果：

```text
A：+5P -10P = -5P
B：+10P
C：+10P
```

Aは複数人から狙われていますが、貯める失敗の -10P は1回のみです。

### 例3：複数人に狙われた守る

```text
A：守る
B：奪う → A
C：奪う → A
```

結果：

```text
A：+5P
B：-5P
C：-5P
```

Aは複数人から狙われていますが、守る成功の +5P は1回のみです。

### 例4：誰にも狙われなかった守る

```text
A：守る
B：貯める
C：貯める
```

結果：

```text
A：-1P
B：+5P
C：+5P
```

## 喧伝システム

ラウンド終了後、希望するプレイヤーは喧伝を行えます。

喧伝を確定すると、選択されたプレイヤーに **+1P** が加算されます。

喧伝したプレイヤーについては、そのラウンドの行動と結果概要が公開されます。

### 喧伝で公開される内容

* 喧伝したプレイヤー名
* そのラウンドの行動

  * 奪う
  * 守る
  * 貯める
* 結果概要

  * 奪うことに成功
  * 守られて失敗
  * 返り討ちに成功
  * 安全に貯めた
  * 貯めたが奪われた
  * 変動なし
  * 空振り など
* 喧伝報酬 +1P

### 喧伝で公開しない内容

* 「奪う」を選んだ場合の対象プレイヤー名

喧伝は、報酬を得る代わりに自分の情報を公開する行為です。

相手に攻撃的なプレイヤー、慎重なプレイヤー、守りを選びやすいプレイヤーといった印象を与えるため、次ラウンドで狙われやすくなる可能性があります。

## オートセーブについて

ゲーム状態はブラウザ内に自動保存されます。

ページを更新したり、ブラウザを閉じたりしても、保存データから再開できます。

ただし、保存データは使用しているブラウザに保存されるため、別のPCや別のブラウザでは共有されません。

## Discord投稿用テキストについて

最終結果画面では、Discord投稿用の結果テキストが自動生成されます。

投稿用テキストには以下の内容が含まれます。

* 最終順位
* 各ラウンドの行動
* 各ラウンドの処理結果
* 喧伝ログ
* ラウンド終了時順位

「Discord用テキストをコピー」を押すと、生成された文章をコピーできます。

## 注意

このツールはゲームマスターの運営補助を目的としたものです。

ゲームそのものを自動進行するものではなく、行動入力や結果確認はゲームマスターが行う前提です。

保存データはブラウザ内に保存されるため、使用端末やブラウザを変更すると引き継がれません。
